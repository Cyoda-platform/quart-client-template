# Functional Requirements for Hello World App

## API Endpoints

### 1. POST /hello
- **Purpose:** Trigger business logic or external data retrieval/calculation.
- **Request:**
  ```json
  {
    "name": "string"  // optional, name to personalize greeting
  }
  ```
- **Response:**
  ```json
  {
    "message": "string"  // greeting message, e.g. "Hello World" or "Hello, {name}!"
  }
  ```

### 2. GET /hello/result
- **Purpose:** Retrieve the latest greeting message generated by the POST endpoint.
- **Request:** No body required.
- **Response:**
  ```json
  {
    "message": "string"  // last greeting message
  }
  ```
